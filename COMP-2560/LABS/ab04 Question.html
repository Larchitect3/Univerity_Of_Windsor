Write a bash script, call it countExtScript.bash, that takes a list extensions and for each extension saves the total number of files it finds in a directory.

Synopsis:  countExtScript.bash  [-t targetDir] [-c countFile] <extension-list>

where options:

-t : specify a target directory (if missing then use current directory .).

-c : specify a text file to save counts. For each extension, the script writes a line consisting of extension name, a space and the file count. When -c is

      missing, counts are printed to the screen.

Call examples:

%> countExtScript.bash -t  ~  -c total.txt  doc pdf jpg     # target directory is home directory (~) and results are to be saved in file total.txt.

%> countExtScript.bash txt pdf jpg                                 # target directory is current directory (.) and results are to be printed on the screen.

%> countExtScript.bash -t oldFiles txt pdf jpg                # target directory is oldFiles and results are to be printed on the screen.
